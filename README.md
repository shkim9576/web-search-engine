# Web Search Engine
Web Search Engine is comprised of 3 components: Web crawler (or splider), Indexing, and Query.
This project implements Google like web search engine that includes crawler and web UI for query.
Indexing and Query part are implemented using Solr.

## Additional Features
In additon to basic web search engine functionality, this project also implements 3 major features.
* Spelling Correction
* Recommended Search
* Selecting Different Ranking Algorithm
